# Homebrew Tap

Homebrew formulae for my projects.

## Installation

```bash
brew tap nozomi-koborinai/tap
brew install gcp-cost-mcp-server
```

|---------|-------------|
| [gcp-cost-mcp-server](https://github.com/nozomi-koborinai/gcp-cost-mcp-server) | MCP server for estimating Google Cloud running costs |

```bash
brew update

```
