# Homebrew Tap

Homebrew formulae for my projects.

## Installation

```bash
brew tap nozomi-koborinai/tap
```

Then install any formula:

```bash
brew install <formula-name>
```

## Available Formulae

| Formula | Description | Status |
|---------|-------------|--------|
| [ableton-osc-mcp](https://github.com/nozomi-koborinai/ableton-osc-mcp) | MCP server for controlling Ableton Live via AbletonOSC | Alpha |
| [gcp-cost-mcp-server](https://github.com/nozomi-koborinai/gcp-cost-mcp-server) | MCP server for estimating Google Cloud running costs | Stable |

## Updating

```bash
brew update
brew upgrade <formula-name>
```

## Notes

- **Alpha** versions may have breaking changes between releases.
- For issues, please open an issue in the respective project repository.
